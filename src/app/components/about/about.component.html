About Agora Vote

Agora is a library of data structures and algorithms for counting votes in elections.
Learn More about Agora

Agora vote is a voting platform where users can create elections and invite friends to cast their votes.
This application uses Agora Web API as backend application.

For contributing to the Agora Web app or reporting any issues, visit https://gitlab.com/aossie/Agora-web-frontend

Agora & Agora Vote is maintained by AIOSSE, Australian Open Source Software and Education and all its contributors.
